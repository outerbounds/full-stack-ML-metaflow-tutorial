Getting started with Argo

This flow looks very similar to the previous one, with the addition of the schedule decorator.

When the flow is run with the following command:
python flows/cloud/tree_branch_cloud_flow_deploy.py --environment=conda argo-workflows create
then the flow is deployed to Argo.

Since the FlowSpec is annotated with the @schedule(cron='58 * * * * *'), this flow will be run on Argo every hour on the 58th minute.

Change this value to a minute value that is coming up, leave the rest of the cron expression go to your Argo UI, and watch to see the flow run.

Now, go to the Argo deployment attached to your sandbox by finding your ID from the URL in your browser. Your sandbox has a unique ID, visible in the UsRL with this pattern. https://vs-YOUR_ID.outerbounds.dev/ Remove the vs part in front, and everything after .dev, then navigate to your sandboxes Argo deployment at: https://argo-YOUR_ID.outerbounds.dev/

Check out the docs linked above to see more patterns.
