The Baseline TitanicSurvivalPredictor Flow

Let's deploy the baseline flow. In the champion/challenger perspective, this model will be considered the reigning champion until we discover a better one to replace it with.

To deploy the flow, run the following command:
python flows/local/titanic_survival_predictor.py --environment=conda --production --branch champion argo-workflows create

To run the flow, run the following command:
python flows/local/titanic_survival_predictor.py --environment=conda --production --branch champion argo-workflows trigger
