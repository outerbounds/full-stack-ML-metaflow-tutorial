The Challenger TitanicSurvivalPredictor Flow

Let's deploy the challenger flow side-by-side with the champion/baseline flow.

To deploy the flow, run the following command:
python flows/local/titanic_survival_predictor_xgb.py --environment=conda --production --branch challenger --production argo-workflows create

To run the flow, run the following command:
python flows/local/titanic_survival_predictor_xgb.py --environment=conda --production --branch challenger --production argo-workflows trigger
